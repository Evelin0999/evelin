Glosario Tecnológico

A - M

A | B | C | D | E | F | G | H | I | J | K | L | M

A

Algoritmo: Conjunto de instrucciones o reglas definidas para realizar una tarea.

B

Bug: Error o defecto en un software que causa un comportamiento inesperado.

C

Compilador: Programa que traduce el código fuente a un lenguaje ejecutable.

D

Depuración: Proceso de identificar y corregir errores en el código.

E

Etiquetas: Elementos de código HTML que definen la estructura de una página web.

F

Framework: Conjunto de herramientas y bibliotecas que facilitan el desarrollo de software.

G

Git: Sistema de control de versiones distribuido utilizado en el desarrollo de software.

H

HTML: Lenguaje de marcado utilizado para crear páginas web.

I

Intérprete: Programa que ejecuta código directamente, sin necesidad de compilarlo previamente.

J

JavaScript: Lenguaje de programación utilizado para crear interactividad en páginas web.

K

Kernel: Núcleo del sistema operativo que gestiona los recursos del hardware.

L

Librería: Conjunto de funciones y procedimientos que pueden ser utilizados por un programa.

M

Markdown: Lenguaje de marcado ligero utilizado para formatear texto.

N - Z

N | O | P | Q | R | S | T | U | V | W | X | Y | Z

N

Netbook: Computadora portátil pequeña y económica.

O

OAuth: Protocolo de autorización que permite el acceso a servicios sin compartir credenciales.

P

Python: Lenguaje de programación de alto nivel y fácil de aprender.

Q

Quicksort: Algoritmo eficiente de ordenación basado en el principio de divide y vencerás.

R

Router: Dispositivo que enruta el tráfico de datos entre redes de computadoras.

S

SQL: Lenguaje de consulta estructurado utilizado para interactuar con bases de datos.

T

Token: Unidad de autenticación utilizada para acceder a un servicio de manera segura.

U

UDP: Protocolo de comunicación sin conexión utilizado en redes.

V

VPN: Red privada virtual utilizada para acceder de manera segura a una red remota.

W

Wi-Fi: Tecnología de comunicación inalámbrica utilizada para redes locales.

X

XHTML: Extensión de HTML que utiliza una sintaxis más estricta basada en XML.

Y

YAML: Lenguaje de marcado utilizado para representar datos estructurados de manera legible.

Z

Zero-Day: Vulnerabilidad de seguridad desconocida en el software antes de ser parcheada.
